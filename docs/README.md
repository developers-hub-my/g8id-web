# Documentation

G8ID Web is the marketing website for [G8ID](https://g8id.com), an on-premise Identity Data Management
system. Built with Astro, Tailwind CSS v4, and Alpine.js. Deployed on Netlify.

Site architecture, design system, and component patterns.
